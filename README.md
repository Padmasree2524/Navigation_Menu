# # 🎓 EduLearn Academy - Education Website

## 📌 Project Overview

EduLearn Academy is a responsive education website designed to provide information about courses, services, and learning opportunities.

This project was developed as part of Task 3: Navigation Menu Implementation.

## ✨ Features

- Responsive Navigation Menu
- Single Page Section Navigation
- Attractive Home Landing Page
- Course Cards
- Services Section
- Why Choose Us Section
- Contact Information
- Smooth UI Animations
- Modern CSS Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
